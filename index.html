Features

Check out TenX Features

Statusbar

Statusbar customisations

We've added the ability to customize the statusbar completely including the icons, height, etc.
Some of the customisations are as follows.

  • Statusbar icons.
  • Custom Logo.
  • Battery Bar.
  • Battery styles.
  • Clock.
  • VoLTE icons.
  • VoWiFi Icons, etc.

Quick Settings

Quick Settings customisations

  • QS Tile columns count.
  • Quick bar columns count.
  • Brightness slider position.
  • Battery modes.
  • System info.
  • Footer text.
  • Footer visibilities, etc.

User Interface

User Interface customisations

  • RGB Accent & Gradient picker.
  • QS Tiles style.
  • Switch style.
  • QS Header image.
  • QS Panel background image.
  • UI style
  • QS Tiles tint, etc.

Screen

Screen customisations

  • Force fullscreen for apps.
  • Rounded corners.
  • Statusbar padding, etc.

Lock screen

Lockscreen customisations

  • Clock styles.
  • Date styles.
  • Item's padding.
  • Weather.
  • FOD icon picker.
  • FOD Animation picker.
  • Charging info, etc.
  • Statusbar padding, etc.

Notifications

Notifications customisations

  • Heads Up.
  • Ambient Edge Lights.
  • Flash flashlight on incoming calls, etc.

Navigation

Navigation customisations

  • Navigation bar tuner.
  • Pilse visualizer, etc.

Animations

Animations customisations

  • System animation.
  • Power menu animations.
  • Screen off animations.
  • Toast animations.
  • QS Tile animations.
  • List view animations.
  • Animations duration, etc.

Miscellaneous

Miscellaneous customisations

  • Gaming mode.
  • Wake on charge.
  • Adaptive playback, etc.
